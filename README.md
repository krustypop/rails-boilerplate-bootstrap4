## To set this up

Clone the repo:
```
git clone https://github.com/krustypop/rails-boilerplate-bootstrap4.git
```


Digg into it:
```
cd rails-boilerplate-bootstrap4
```


Install all gems:
```
bundle install
```


To infinity and beyond:
```
rails server
```


That's it !!
